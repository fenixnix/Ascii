## 剧本创作流程：

* 故事梗概：内容精简到3~5行，需要准备多个副本可以挑选，从中摸索更好的内容，并不断修改，将精华慢慢凝聚到一个情节中
首先定义主角，配角根据情节需要创作

* 基础情节：游戏情节依附于游戏机制，机制和情节因该相互配合，剧情协助说明游戏机制，机制推动情节。

* 充实内容：突出 序破急-起承转合
    - 序：开端
    - 破：高潮之前的部分，情节持续提升展开
    - 急：高潮和结局，爆发性的展现，让情感释放然后收尾

    * 起：开端
    * 承：发展
    * 转：发生事件
    * 合：结局

### 开头：游戏开头，引子
游戏剧本面向对象是游戏开发人员而不是终端用户
剧本的格式来展现情节
时间，地点，人物，话语，内心，旁白，画面表现，BGM，音效

## 3-1 角色定位
分配给故事中各个角色立场

避免角色重复

**定位重复会以角色重复的形式体现出来**

### 与系统直接关联的定位
* 主人公：一般为玩家操作角色。主人公角色的行动和选择尽量尊重玩家意志。
    - 玩家分身
    - 有代入感：没有特征的方式描绘
        - 刻划不体现个性
        - 增加玩家可选择余地
    - 换位型&角色扮演型：DQ&FF
        - 换位型：尽可能贴近实际玩家
        - 角色扮演型：玩家自主贴近主人公
* 待攻克对象：体现游戏目的,e.g.:侦探游戏中的犯人，王道游戏中的魔王。阶段性待攻克对象，最终目标。
    * 同一时间待攻克对象不宜过多。
    * 待攻克对象应设定细致，描写要有魅力
* 系统角色：与游戏系统进行交互的角色。e.g.:存档，买卖,教程。保证游戏自然顺畅
    * 引导员：向玩家介绍游戏系统及玩法
    * 特殊用法：一般系统角色定位不会发生太大变化，但有时可以让其放弃系统角色切换到其他定位中。e.g.：旅店老板是杀手。战斗系统引导员是BOSS
### 与主人公对立的定位：*让角色对立便于我们创作故事*
* 敌人角色：妨碍主角达成目的，还可以与主人公有相同的目的从而形成竞争*冲突*。从而使故事更加有趣。敌人的描绘会凸显主角印象
    * 恶人角色：绝对恶：***任何人看了都觉得邪恶的角色***。设定标准要以现实世界为基准，再以世界设定里的价值观作为补充和强化。
        * 外观：可怖的外表（容貌，衣着），设定反差
    * 敌对角色：与主角敌对，看起来恶
* 难关角色：必须攻克的难关，故事推进的必要条件。攻克后可转变为同伴
* 竞争者角色：主角的衬托，相对主角：不相上下,身份对立，与主角竞争成长。更直观得体现主角的成长。与主角有相同的目的。
### 与主人公同阵营的定位
* 情侣：帮手&负担，能够提升玩家达到目的的动力；提升情侣角色对主角动力的方法：给情侣安排特殊境遇；让情侣角色保守秘密；为情侣角色和主角设置特殊关系；在情侣角色与主人公间设置障碍。最好设置一个对情侣相反的角色来作为情侣的衬托，甚至设置与情侣竞争主角。游戏开始如果已经确立关系，则需要说明其经过。负担：枷锁，作为人质，产生纠葛
* 伙伴：
    * 相同目的的战友：提供主角所不具备的能力的补充
    * 倾听者：为了不让角色自言自语，安排与角色的对话来体现主角的内心活动。给主角以鼓舞。***引出主角感情，增强感情振幅，加强玩家理解***
    * 衬托：衬托大雄的铜锣卫门。***用自身魅力衬托主角，以主角相反的性质衬托主角***

    成为伙伴的过程：以敌对身份登场，成为伙伴前充分展现其魅力（让玩家想将其收为伙伴）
    
    避免好好先生：尽量与主人公对立，让故事更加有趣，体现角色个性，创造故事契机

    死角色：故事中失去必要性，避免死角色：*牺牲&变脸*

    <u>为敌人安排伙伴角色</u>

* 家人:血缘或者婚姻关系
    * 基于社会上”家人是好的“这一基本观点。
    * 以社会基本观念为基础给角色套上枷锁（可以是不同阵营的角色）效果极强的定位
    * 家人的存在感会淡化

### 给故事带来变化的定位：避免故事变成一条直线
* 契机角色：能给主角意志、行动、感情带来变化（通常是解决事件的委托）
* 救星
* 叛徒
* 贤者
### 补充和强化故事的定位
* 缓冲角色
* 深论角色
* 动物角色
* 妹妹角色
* 闲杂角色

***一个角色可以有多个定位。越是重要的角色定位就越多。***

比如 女性‘伙伴’角色兼任‘贤者’为主角指点迷津，同时作为主人公恋爱‘待攻克对象’

随着故事发展，角色定位会发生变化

比如：比克大魔王：恶人->伙伴->贤者

*角色的定位像面具般可以替换，也可以一次戴好几个*

# 4世界：从何时=时间；何地=地点
+ 时间
    - 什么时代：过去；未来；现代
    - 公历多少年
    - 这个时代的特征
+ 地点
    - 主要舞台
    - 哪个国家
    - 地区
    - 地方特征
+ 世界状态
    - 和平or动荡，为什么
    - 国家或类似概念:多少国家，主要国家
    - 宗教或类似概念:是否能影响世界，有哪些宗教，主要宗教 
    - 民族：是否只有一个民族，主要民族有哪些
+ 社会系统
+ 家族制度
+ 民生&一般社会理念
+ 世界法则
+ 语言
+ 交通手段
+ 通信手段

## 创建怎么样的世界
参考主题与故事
+ 如果世界是这样的呢？
+ 制定世界规则
    - 规则与真实感：*只要让玩家接受，不管多么非现实，多么无稽，玩家都不会感到别扭。为此必须明确设定世界的真实感，创作一个具有始终如一法则的世界，规则才能运作才能被玩家接受*
    > 创作有规则的世界
 
    > 创作大众能接受的规则
+ 世界存在于游戏系统中：*世界的详细规则建立在详细的游戏系统之上。创作游戏剧本时应该掌握游戏系统*

## 世界与角色、故事的关系
+ 以世界为起点创作角色&故事
+ 主题 故事 角色 世界是相互关联的

# 5 结构
 + 为什么剧本写不到最后：失去兴致，故事出现纰漏
 + 会写文章不一定会写剧本：
 + 结构是防止摔倒的拐杖

 > 先确立结构再写
 + 音乐也有结构
 + 前奏和副歌最先被人记住的理由：创作音乐时，故意将前奏和副歌做得最容易给人留下印象的组合方式
 > 传递信息时，最容易给人留下印象的组合方式
 + 音乐结构与剧本的结构
    - 音乐的基本结构：前奏，旋律A，旋律B，副歌，尾声
 
 前奏吸引注意力，旋律A向副歌铺垫情绪，旋律B将情绪压缩，副歌释放情绪，尾声收尾。
 这组合中最重要的就是前奏和副歌

 + 音乐->剧本
 + 前奏->开端：在开端抓住玩家的心
 + 旋律A->通往高潮的过程：如何让玩家保持在饥饿状态
 + 旋律B->通往高潮的过程
 + 副歌->高潮：在高潮不恩表达出最想表达的东西，创造激动人心的时刻
 + 尾声->结局 

***结构就像音乐的乐谱***

### 结构的种类
+ 结构的形式：不止是起承转合
    - 起承转合：四格漫画
        - 起：事物开端
        - 承：发展
        - 转：转折
        - 合：结论
    - 序破急：源自雅乐
    - 三幕式结构：希腊悲剧，莎士比亚，TV剧
        - 开端
        - 过程
        - 结局
+ 本书对结构的观点
    - 开头
    - 发展
    - 结尾
---
> 一个具体例子
> - 开头：某人前往地域讨伐恶魔
> - 发展：某人与恶魔展开卓绝的战斗
> - 结尾：某人击败了恶魔，创造了没有地狱的世界

+ 分支：游戏剧本有额外必须考虑的东西，那就是游戏失败和选项。也就是剧本上的分支。

+ 活用神话：神话的故事模式（千面英雄）
 - 英雄遭遇挫折，受贤者指引，击败强大敌人的故事
 - 主角拔出相传无人能拔出的石中宝剑，证明自己就是英雄
 - 英雄消灭盘踞地下城的魔物，从而获得宝物的故事
 - 主角前往黄泉国度冒险的故事
 - 哥哥杀死比自己优秀的弟弟，孩子为信念杀死父母的故事

# 6 结构的要素
### **开端**:游戏的开头，抓住玩家的心
    - 传达游戏目的：玩家在这款游戏里该做什么
    - 传达游戏玩法：展示游戏玩法，游戏教程
    - 传达这是个什么样的游戏：通过开端了解主要角色，角色关系，展示世界观
### **通往高潮的过程**：游戏中最长的部分，保持一定长度和趣味性，始终保持饥饿感。游戏的主体
    - 设定游戏的失败：loss is fun，游戏失败在故事中的意义由编剧创作
+ 章节分四类：通往结局的过程的一个个条件过程就是章节
打倒魔王:获得火把->去消灭龙->打倒龙
    - 达成游戏目的的章节：
        - 给目的设置障碍的章节 
        - 给目的提供援助的章节
    - 推进剧本的章节：推进剧情的契机，案件，事件
    - 加深剧本主题的章节：凸显剧情主题，使其兼具推进剧本的章节
    - 偏离游戏本来目的的额外章节:丰富游戏内容，提供支线任务，小游戏。补充说明世界，获取额外奖励
### **高潮**：剧烈，兴奋，纠葛的最高峰，对立的最高峰
***达成游戏目的的瞬间***
- 通往高潮的部分一定要有趣
    - 铺垫和发展：动用全员角色，用上之前所有的游戏机制
高潮的成败由通往高潮的过程左右
> 高潮：前面整个过程的总结算

### **结局**
结局向玩家传达游戏目的达成后产生了什么结果的部分
- 完美结局
- 序作伏笔

设定游戏结束：
 - 游戏完全结束
 - 告诉玩家还没有看到真正结局（多结局游戏）

**重玩**：
- 重玩与游戏机制有着斩不断的关系，重玩机制可以让游戏剧本做出一些比较大胆的决定。
- 在失败时让玩家期待成功。过多的失败也会让玩家感到厌恶。
- 游戏失败能强制让玩家熟悉游戏规则及机制
